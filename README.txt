
Alien Invasion (Python + Pygame)

Structure:
 ┣ src/
 ┃ ┣ alien_invasion.py         → Main entry point
 ┃ ┗ alien_invasion_main.py    → Full class-based game logic
 ┣ images/                     → For future graphics
 ┣ sounds/                     → For future sounds
 ┗ README.txt                  → This file

Usage:
  pip install pygame
  python src/alien_invasion.py

Controls:
  ← / → or A / D = Move
  Space = Fire bullet
  P or click Play = Start / Restart
  Q or ESC = Quit
 
 cách chạy:
 python src/alien_invasion.py
